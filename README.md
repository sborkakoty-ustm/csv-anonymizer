# CSV Anonymizer (Flask)

A simple Flask web application that allows users to upload a CSV file, select columns to anonymize, and download an anonymized version of the dataset.

## Features
- Upload CSV files
- View dataset attributes and data types
- Select columns for anonymization
- Pseudonymize text data
- Bin numeric data
- Download anonymized CSV

## Tech Stack
- Python
- Flask
- Pandas
